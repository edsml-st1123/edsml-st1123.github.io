◆ apply_projection()

Image Utility::apply_projection ( Volume & vol)
staticprivate

Allows the user to choose between different projection method to use.

Parameters
volThe volume to apply projection.

This function helps the user apply projection to the volume, allowing them to choose between Maximum Intensity Projection, Minimum Intensity Projection, and Mean Intensity Projection. The user can also choose between Gaussian and Median filters, or no filter at all. The user can also choose the kernel size for the filter. The function returns the image after applying the projection.
