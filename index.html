Welcome to Advanced Programming Image Processing Project - Group Linear Regression

This project aims to provide a comprehensive framework for image processing tasks, focusing on linear regression techniques. Whether you're a beginner or an expert in image processing, this project offers a wide range of functionalities to handle various aspects of image analysis and manipulation, including 2D and 3D image processing, filtering, projection, and slicing.

Features:

  • 2D and 3D Image Processing: Supports both 2D and 3D image manipulations.
  • Comprehensive Filters and Projections: Includes a variety of filters such as Grayscale, Brightness, Histogram Equalisation, and Edge Detection, along with 3D data volume filters and orthographic projections.
  • User-friendly Interface: Easy interaction with the project's functionalities.
  • Extensible Architecture: Designed with modularity in mind for easy extension.

Getting Started:

To get started with the project, follow these steps:

  1. Clone the repository to your local machine.
  2. Explore the provided functionalities in the src directory.
  3. Refer to the documentation for detailed information on each module and class.
  4. Start experimenting with image processing tasks using the provided utilities.

Prerequisites:

Ensure you have a C++17 compliant compiler installed. This project uses features introduced in C++17, so compatibility with this or later standards is required.

Installation:

  1. Clone the repository to your local machine:
    git clone https://github.com/ese-msc-2023/advanced-programming-group-linear-regression.git
  2. Navigate to the project directory:
    cd advanced-programming-group-linear-regression
  3. Build the project:

Using g++:

g++ -std=c++20 -o main src/main.cpp src/filter/filter.cpp src/Image/Image.cpp src/projection/projection.cpp src/slice/slice.cpp src/utility/utility.cpp src/volume/volume.cpp
./main

Using CMake:

mkdir build && cd build
cmake ..
make

Running the Application: You can run the executable provided in the build directory. Use the following command to start processing images or data volumes:

./main_win # for win
./main_mac # for mac

Testing:

To run the unit tests included with this project, navigate to the root directory and execute the test suite:

./test_program # needs to change to the final name

Or build the testing framework yourself:

g++ -std=c++20 -o test src/test_framework.cpp src/filter/filter.cpp src/Image/Image.cpp src/projection/projection.cpp src/slice/slice.cpp src/utility/utility.cpp src/volume/volume.cpp
./test
