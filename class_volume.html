void Volume::save ( const std::string & directoryPath)

Saves the volume to the specified directory.

Parameters
directoryPathThe path to the directory to save the volume.

Saves the volume to the specified directory. The images are saved as PNG files with filenames image0.png, image1.png, etc. The images are saved in the order they were loaded. If the directory does not exist, an error message is printed to the console. If an image fails to save, an error message is printed to the console.
