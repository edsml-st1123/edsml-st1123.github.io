static

Computes the Average Intensity Projection (AIP) from the given Volume.

Parameters
volThe Volume object from which to generate the AIP.
filter_methodThe method used for filtering (default is 3).
kernelSizeThe size of the kernel used for filtering (default is 7).
Returns
The AIP image.

This function computes the Average Intensity Projection (AIP) from the given Volume. Volume is a class that contains a vector of Image objects. The AIP is computed by averaging the pixel values of each Image in the Volume.
