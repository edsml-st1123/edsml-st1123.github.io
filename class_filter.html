◆ findMedian()

unsigned char Filter::findMedian ( std::vector< unsigned char > & neighborhood)
staticprivate

Finds the median value from the given neighborhood.

Parameters
neighborhoodThe neighborhood vector.
Returns
The median value.

Helper function to find the median value from the given neighborhood. The neighborhood is a vector of unsigned chars. The median value is the middle value when the neighborhood is sorted in ascending order. If the neighborhood size is even, the median is the average of the two middle values.
